Block Buster Editor

Description

Block Buster Editor is the Map Editor for the upcoming voxel first-person shooter Block Buster. With this tool, blocks and ramps can be placed to create interesting shooter maps. The first design goal of this program is simplicity to use, but it also offers powerful utilities to ease and speed up the creation process.

Tools

Basic Tools
Place

With this tool you can place block or ramps on the map. To do so, simply left click on one of the faces of a block/ramp that has already been placed on the map. You can also remove existing blocks with this tool by right clicking on them.
